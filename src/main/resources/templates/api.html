1. Εισαγωγή

Η εφαρμογή μας υποστηρίζει ένα RESTful Application Programming Interface (REST API) για την ανάκτηση, αναζήτηση, καταχώριση και επεξεργασία των δεδομένων της βάσης μας. Σκοπός μας η διεύρυνση των δυνατοτήτων της εφαρμογής, προσφέροντας την δυνατότητα στο ευρύ κοινό για χρήση της πληροφορίας που έχει συγκεντρωθεί, μέσω της δράσης εθελοντών χρηστών και επιχειρήσεων. Καθένας που επιθυμεί να χρησιμοποιήσει την εφαρμογή μας σε επίπεδο ανάπτυξης δικής του πλατφόρμας και αντλώντας πληροφορία από την βάση μας, είναι πλέον σε θέση να το κάνει, μέσω του διατιθέμενου API.

2. Δικαιοδοσία Χρήστη API

Οι δυνατότητες που παρέχονται στο χρήστη του API που διαθέτει η εφαρμογή διαφοροποιούνται, ανάλογα με το ρόλο του κάθε χρήστη στην εφαρμογή μας και τα δικαιώματα που καθορίζονται από αυτόν.

2.1. Guest Χρήστης

Κάθε χρήστης τύπου Guest (μη εγγεγραμμένος) έχει τη δυνατότητα ανάκτησης και αναζήτησης δεδομένων μέσω του API. Μπορεί επομένως, μέσω συγκεκριμένων κλήσεων της HTTP μεθόδου GET, να λάβει τις εξής πληροφορίες, σε json μορφή:

  • GET https://localhost:8765/observatory/api/products
    Επιστρέφεται η λίστα των προϊόντων του παρατηρητηρίου.
  • GET https://localhost:8765/observatory/api/products/{id}
    Επιστρέφονται τα δεδομένα του προϊόντος με το συγκεκριμένο id.
  • GET https://localhost:8765/observatory/api/shops
    Επιστρέφεται η λίστα των καταστημάτων του παρατηρητηρίου.
  • GET https://localhost:8765/observatory/api/prices
    Αναζήτηση τιμών προϊόντων σε καταστήματα.

2.2. Subscribed Χρήστης

Κάθε χρήστης τύπου Subscribed (εγγεγραμμένος) έχει τη δυνατότητα ανάκτησης, αναζήτησης, καταχώρισης και επεξεργασίας δεδομένων μέσω του API. Μπορεί επομένως, μέσω συγκεκριμένων κλήσεων των HTTP μεθόδων GET, POST, PUT, PATCH και DELETE να λάβει τις εξής πληροφορίες, σε json μορφή:

  • GET https://localhost:8765/observatory/api/products
    Επιστρέφεται η λίστα των προϊόντων του παρατηρητηρίου.
  • POST https://localhost:8765/observatory/api/products
    Δημιουργία νέου προϊόντος.
  • GET https://localhost:8765/observatory/api/products/{id}
    Επιστρέφονται τα δεδομένα του προϊόντος με το συγκεκριμένο id.
  • PUT https://localhost:8765/observatory/api/products/{id}
    Ενημέρωση των στοιχείων του προϊόντος με το συγκεκριμένο id.
  • PATCH https://localhost:8765/observatory/api/products/{id}
    Ενημέρωση ενός μόνο πεδίου του προϊόντος με το συγκεκριμένο id.
  • DELETE https://localhost:8765/observatory/api/products/{id}
    Διαγραφή του προϊόντος με το συγκεκριμένο id.
  • GET https://localhost:8765/observatory/api/shops
    Επιστρέφεται η λίστα των καταστημάτων του παρατηρητηρίου.
  • POST https://localhost:8765/observatory/api/shops
    Δημιουργία νέου καταστήματος.
  • GET https://localhost:8765/observatory/api/shops/{id}
    Επιστρέφονται τα δεδομένα του καταστήματος με το συγκεκριμένο id.
  • PUT https://localhost:8765/observatory/api/shops/{id}
    Ενημέρωση των στοιχείων του καταστήματος με το συγκεκριμένο id.
  • PATCH https://localhost:8765/observatory/api/shops/{id}
    Ενημέρωση ενός μόνο πεδίου του καταστήματος με το συγκεκριμένο id.
  • DELETE https://localhost:8765/observatory/api/shops/{id}
    Διαγραφή του καταστήματος με το συγκεκριμένο id.
  • GET https://localhost:8765/observatory/api/prices
    Αναζήτηση τιμών προϊόντων σε καταστήματα.
  • POST https://localhost:8765/observatory/api/prices
    Προσθήκη τιμής προϊόντος.
